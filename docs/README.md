## Quick Start Guide

### For Business Owners (Direct Customers)

1. **Install WordPress plugin** → 5-minute setup
2. **Connect Google Ads** → OAuth, simple authorization
3. **Fill out your details** → Business vertical, average customer value
4. **Enable tracking** → Forms, calls, calculator interactions
5. **Watch results** → Leads automatically scored and synced to Google Ads

**Value**: See which ads actually drive customers in 30 days vs. paying attention to clicks and form fills.

### For Agencies

1. **Sign up for AdVocate Agency account** → Multi-client dashboard
2. **Install plugin on client sites** → Branded or standard version
3. **Configure optimization rules** → Budget shifts, pausing, alerts
4. **Monitor all clients** → Unified view across Google Ads + Meta
5. **Scale efficiently** → Handle more clients without more people

**Value**: Automate the tedious work, focus on strategy, charge for results not hours.
