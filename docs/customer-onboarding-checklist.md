# AdsEngineer Customer Onboarding Checklist

## Pre-Onboarding (Your Tasks)
- [ ] Send welcome email with pricing confirmation
- [ ] Create customer account in your system
- [ ] Send Shopify integration guide
- [ ] Schedule onboarding call (optional)

## Customer Setup (Their Tasks)

### 1. Account Setup
- [ ] Log into AdsEngineer dashboard
- [ ] Verify account details and billing
- [ ] Choose plan: Starter (€80/mo) or Professional (€250/mo)

### 2. Shopify Integration
- [ ] Provide Shopify store URL
- [ ] Grant AdsEngineer app permissions (if using automatic setup)
- [ ] Or manually install tracking snippet (see integration guide)
- [ ] Verify snippet is loading on website

### 3. Google Ads Setup
- [ ] Connect Google Ads account via OAuth
- [ ] Select or create conversion action
- [ ] Verify connection status shows "Active"
- [ ] Test conversion tracking (optional)

### 4. Meta Ads Setup (Professional Plan Only)
- [ ] Connect Meta Business account
- [ ] Select Facebook Pixel
- [ ] Verify Meta connection status
- [ ] Test Meta conversion tracking (optional)

### 5. Testing & Verification
- [ ] Place test order on Shopify store
- [ ] Check AdsEngineer dashboard for tracking
- [ ] Verify conversions appear in Google Ads (1-2 hours delay)
- [ ] Verify conversions appear in Meta Ads (Professional plan)

## Post-Onboarding (Your Tasks)

### Day 1: Welcome & Setup
- [ ] Confirm all integrations are working
- [ ] Send setup confirmation email
- [ ] Provide customer success contact info
- [ ] Schedule check-in call for Week 1

### Week 1: Monitoring
- [ ] Monitor conversion tracking performance
- [ ] Check for any setup issues
- [ ] Send weekly performance summary
- [ ] Answer any customer questions

### Month 1: Optimization
- [ ] Review conversion data and ROI
- [ ] Provide optimization recommendations
- [ ] Check customer satisfaction
- [ ] Prepare for price increase (if applicable)

## Troubleshooting Checklist

### Common Issues:
- [ ] Snippet not loading: Check theme code installation
- [ ] Webhooks failing: Reinstall Shopify webhooks
- [ ] Conversions not appearing: Wait 1-2 hours, check credentials
- [ ] Dashboard not updating: Clear browser cache, refresh

### Support Response Times:
- Critical issues: < 2 hours
- Setup help: < 4 hours
- General questions: < 24 hours

## Success Metrics
- [ ] Tracking snippet active on website
- [ ] Google Ads connection verified
- [ ] Meta Ads connection verified (if applicable)
- [ ] Test conversions appearing in ad platforms
- [ ] Customer reports seeing ROI

## Pricing Reminder
- **Starter Plan**: €80/month (Google Ads only)
- **Professional Plan**: €250/month (Google + Meta Ads)
- **Beta Period**: First 3 months at discounted rate
- **Setup Fee**: €0 for beta customers

## Customer Handoff
- [ ] All integrations working
- [ ] Customer trained on dashboard
- [ ] Support contact provided
- [ ] Next check-in scheduled
- [ ] Documentation provided
