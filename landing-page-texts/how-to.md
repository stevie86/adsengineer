## Setup Steps Section

### Section Title
Setup in 3 Steps

---

### Step 1

**Title**
Install WordPress Plugin

**Description**
Download and activate the AdsEngineer plugin from the WordPress repository. Compatible with Contact Form 7, Gravity Forms, and any custom setup.

**Features**
- ~5 minutes to install
- No coding required
- GCLID capture enabled by default

---

### Step 2

**Title**
Connect Your Data Source

**Description**
Choose your integration and provide webhook URLs or API keys. We handle the rest automatically.

#### GoHighLevel
- Add webhook URL in Workflows
- Trigger on: Lead Created, Deal Won
- Include: Pipeline, Deal Value

#### Shopify
- Add webhook in Settings → Webhooks
- Trigger on: Order Created, Order Paid
- Include: Order Total, Customer LTV

#### Custom Forms
- Point form to AdsEngineer endpoint
- Any HTTP POST supported
- GCLID passed automatically

---

### Step 3

**Title**
Configure Google & Meta

**Description**
Add AdsEngineer's conversion tracking to your ad platforms. Revenue and lead quality start flowing back immediately.

#### Google Ads
- Enable Enhanced Conversions
- Import GCLID with each conversion
- Add revenue value to bid signals
- Use value-based bidding strategies

#### Meta CAPI
- Set up Conversions API
- Add server-side event source
- Send purchase value data
- Configure deduplication

## What Happens Next Section

### Section Title
What Happens Next

### Automatic Sync
Every time a lead is created, deal won, or order placed, AdsEngineer automatically sends that data to Google Ads and Meta. No manual uploads, no spreadsheets.

### Value-Based Bidding
Google's and Meta's algorithms learn which clicks become high-value customers. Your ads automatically shift budget toward better-performing campaigns and audiences.

### Dashboard & Analytics
Track conversion rates, revenue per campaign, and lead quality scores in one dashboard. See exactly which campaigns and keywords are driving real customers.
